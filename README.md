# dreamOS

**Live Demo:** [https://dream-os-psi.vercel.app/](https://dream-os-psi.vercel.app/)

## Overview
dreamOS is a personal productivity application designed to streamline daily workflows. The core feature of the platform is its integration with the Gemini API, providing native AI-driven assistance and text processing directly within the app. 

## Tech Stack
* **Frontend:** React, TypeScript
* **Build Tool:** Vite
* **AI Integration:** Gemini API
* **Deployment:** Vercel

## Key Features
* **AI-Powered Assistance:** Interacts directly with the Gemini API to handle complex natural language queries and improve user productivity.
* **Immersive OS Environment:** A beautifully designed, glassmorphism-inspired interface featuring dynamic themes, custom wallpapers, and interactive window modules.
* **Integrated Productivity Suite:** Includes specialized modules like a Library for PDF reading, a Theory Lab for interactive chess study, and Sonic Scapes for ambient audio.
